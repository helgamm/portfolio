Project 2.1 - Kerf Estimation

Estimating the kerf

The objective of this project was to design a pressfit model and manufacture it using a laser cutter

We began by doing a small group project to estimate the kerf of the laser cutter using material tests. We began by drawing a very simple box with 16 smaller parts inside it using Fusion 360. Then we exported the drawing and uploaded it in Inkscape for printing.

After the model had been printed we took all of the 16 parts out, put them together and measured the total length of them using a calliper. Then we measure the total gap it had created. Then the kerf was calculated by subtracting the pieces inside by the gap and divide the rest by 17 (because there are 17 lines for the 16 pieces). The kerf was estimated to be around 0.17mm.

Project 2.2 - Making a sticker

Sticker using a vinyl cutter

The objective of this project was to design a sticker and print it in a vinyl cutter

For this part of the project we got to make a sticker and cut it using a vinyl cutter. I wanter to make sticker that I could put on my airpods case to make it more easily recognizeable. I began by going onto Canva which is a website where you can easily make designs online. I made a new file and chose a sticker format so the size wouldn't be too big. I decided to go with the letter H, so I chose the font I wanted, typed the letter and downloaded the picture. Then I opened the picture in Inkscape were I was able to add outlines around the picture and after that I could delete the picture itself and was left with just the vectorized outline. I then chose a color of the vinyl and printed the sticker.
