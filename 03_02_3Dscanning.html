Project 3.1

The objective of this project was to make a 3D scan of an object using photogrammetry

For 3D scanning I used the smartphone application Scaniverse. I picked an object, and then scanned it through the app. I was then able to upload the file to Scaniverse and they made it easy to embed the model to this website.
